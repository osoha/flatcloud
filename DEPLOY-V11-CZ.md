# Nasazení FlatCloud Rent V11 RC1

1. Rozbalte ZIP a nahrajte jeho obsah do kořene stávajícího GitHub repozitáře.
2. Commitněte změny do větve `main`.
3. Render automaticky spustí build a deploy.
4. Databázová migrace není potřeba.

## Kontrolní scénář

- otevřít nemovitost a seznam jednotek,
- kliknout na číslo jednotky,
- zkontrolovat vlastníka, nájemníka a smlouvu,
- vytvořit měsíční předpis přímo v detailu jednotky,
- otevřít jeho detail a případně jej vypnout,
- změnit vlastníka jednotky v její editaci,
- změnit hlavního vlastníka / SVJ v záložce Vlastníci,
- zkontrolovat kontaktní kartu správce.
